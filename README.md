# tcalc
basic argv based calculator written in c++.

## Usage
```sh
$ ./main.o 5 + 5
5 + 5 = 10
```

## Operators

| +    | -    | *    | /    | ^    |
|---------------- | --------------- | --------------- | --------------- | --------------- |
| sum + sum    | sum - sum    | sum * sum    | sum / sum    | sum ^ sum   |
